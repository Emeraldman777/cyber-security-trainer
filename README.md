# CyberGuard

CyberGuard is a React + TypeScript + Vite MVP for cyber fraud awareness training. Users classify realistic situations, choose actions, receive consequences, and get a digital vulnerability index from 0 to 100 with a risk profile and recommendations.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- lucide-react
- No backend

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
